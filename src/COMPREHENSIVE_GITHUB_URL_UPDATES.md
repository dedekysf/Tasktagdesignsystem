## üìã Update Pattern for Each Section

For each section above, perform these 2 changes:

### Step 1: Update tabs array
**Find:**
```tsx
tabs={[
  { value: 'preview', label: 'Preview' },
  { value: 'usage', label: 'Usage' }
]}
```

**Replace with:**
```tsx
tabs={[
  { value: 'preview', label: 'Preview' },
  { value: 'usage', label: 'Usage' },
  { value: 'github', label: 'GitHub URL' }
]}
```

### Step 2: Add GitHub TabPanel after usage TabPanel
**Add this code block after the usage TabPanel:**
```tsx
<TabPanel value="github" activeTab={COMPONENT_TAB_NAME}>
  <CodeExample
    title="GitHub URL"
    code={`GITHUB_URL`}
  />
</TabPanel>
```

**Replace:**
- `COMPONENT_TAB_NAME` with the actual tab state name (e.g., `checkboxBasicTab`)
- `GITHUB_URL` with the URL from the mapping above

## üìù Example: Checkbox Basic

### Before:
```tsx
<TabsContainer
  activeTab={checkboxBasicTab}
  onTabChange={setCheckboxBasicTab}
  tabs={[
    { value: 'preview', label: 'Preview' },
    { value: 'usage', label: 'Usage' }
  ]}
>
  <TabPanel value="preview" activeTab={checkboxBasicTab}>
    {/* preview content */}
  </TabPanel>
  <TabPanel value="usage" activeTab={checkboxBasicTab}>
    <CodeExample title="Usage Example" code={`...`} />
  </TabPanel>
</TabsContainer>
```

### After:
```tsx
<TabsContainer
  activeTab={checkboxBasicTab}
  onTabChange={setCheckboxBasicTab}
  tabs={[
    { value: 'preview', label: 'Preview' },
    { value: 'usage', label: 'Usage' },
    { value: 'github', label: 'GitHub URL' }
  ]}
>
  <TabPanel value="preview" activeTab={checkboxBasicTab}>
    {/* preview content */}
  </TabPanel>
  <TabPanel value="usage" activeTab={checkboxBasicTab}>
    <CodeExample title="Usage Example" code={`...`} />
  </TabPanel>
  <TabPanel value="github" activeTab={checkboxBasicTab}>
    <CodeExample
      title="GitHub URL"
      code={`https://github.com/dedekysf/Tasktagdesignsystem/blob/main/components/Checkbox.tsx`}
    />
  </TabPanel>
</TabsContainer>
```
